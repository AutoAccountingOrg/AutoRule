## 简介
该工具是自动记账App的衍生工具，用于构建自动记账的自动分类规则和自动识别规则。

## 更新频率

- 每月一次全量更新。


## 构建

- 基础环境：`NodeJS` 
- 包管理器：`yarn`
- 构建工具：`rollup`
- 代码风格: `eslint`
- 测试工具: `jest`
- 依赖安装：`yarn install`

使用如下命令构建自动记账规则：
```shell
yarn build
```

使用如下命令构建并测试自动记账规则：
```shell
yarn test
```

> [!IMPORTANT]
> 如需发起PR，请确保通过所有测试。

## 使用
- 自动记账的规则会进行周期性更新，自动记账用户跟随App更新提醒进行周期性更新即可。
